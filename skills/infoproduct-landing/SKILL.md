# Продающий лендинг для инфопродукта

## Overview

Методология создания высококонверсионных лендингов для инфопродуктов на рынке СНГ 2026.
**Один лендинг = один продукт.** Структура зависит от ценового сегмента.

**Hero-секция определяет 80% успеха** — у посетителя 3 секунды на понимание предложения.

## When to Use

- Создание лендинга для курса, наставничества, консультации
- Редизайн существующего лендинга инфопродукта
- Брифование структуры для дизайнера

## Интеграция с другими skills

1. **infoproduct-landing** определяет СТРУКТУРУ (какие блоки, в каком порядке)
2. **infoproduct-landing** предоставляет ШАБЛОНЫ секций с конверсионными элементами
3. После структуры вызывай **frontend-design** для стилизации:
   - Средний чек → энергичный/динамичный дизайн
   - Высокий чек → Quiet Luxury (минимализм, воздух)

## Quick Reference: Тип продукта → Структура

| Тип продукта | Экраны | Цель | Ключевые блоки |
|--------------|--------|------|----------------|
| **Курс** (20-50К) | 8-12 | Прямая продажа | Программа, кураторы, сертификат |
| **Интенсив** (5-15К) | 5-7 | Быстрая продажа | Дата, результат за X дней |
| **Наставничество** (100К+) | 15-25 | Запись на диагностику | Кейсы, личность эксперта |
| **Консультация** (5-30К) | 3-5 | Форма/календарь | Кто, что решает, сколько |

## Блоки лендинга (порядок)

### 1. Hero-секция (ОБЯЗАТЕЛЬНО)

**Цель:** За 3 секунды показать ЧТО, ДЛЯ КОГО, КАКОЙ РЕЗУЛЬТАТ

**Формула заголовка:** `[Результат] + [для кого] + [за счёт чего]`

```
❌ "Курс по маркетингу"
✅ "Выйди на доход 150К+/мес в маркетинге за 3 месяца через системный подход к клиентам"
```

**Элементы:**
- [ ] Заголовок (формула выше)
- [ ] Подзаголовок (1-2 предложения — усиление)
- [ ] CTA от первого лица: "Хочу научиться" > "Записаться"
- [ ] Социальное доказательство (мини): "2000+ выпускников" или "Рейтинг 4.9"
- [ ] Визуал: видео/фото эксперта или продукта

**Шаблон:** `section-templates/hero-section.tsx`

---

### 2. Актуализация боли (ВАЖНО)

**Цель:** Эмпатия, не давление. Показать, что понимаешь ситуацию.

**Формат:** 3-5 пунктов "Знакомо?"

```
✅ "Вы проходите курсы, но не можете применить знания"
✅ "Клиенты есть, но доход не растёт — не понимаете, что делать"
❌ "Вы неудачник без нашего курса" (давление)
```

**Шаблон:** `section-templates/pain-points.tsx`

---

### 3. Для кого продукт (ВАЖНО)

**Цель:** Посетитель должен узнать себя в одном из портретов

**Формат:** 3-4 портрета ЦА с конкретикой

```
✅ "Маркетолог с опытом 1-3 года, застрял на 80К/мес"
✅ "Предприниматель, делает всё сам, нет системы"
❌ "Все, кто хочет развиваться" (размыто)
```

**Шаблон:** `section-templates/target-audience.tsx`

---

### 4. Программа/Метод (ВАЖНО для курсов)

**Цель:** Показать системность и путь к результату

**Формат:** Интерактивные раскрывающиеся списки (accordion)

**Структура модуля:**
- Название модуля (результат, не процесс)
- 3-5 тем
- Результат модуля
- Длительность

```
✅ "Модуль 3: Первые клиенты за 2 недели"
❌ "Модуль 3: Продажи" (непонятно что внутри)
```

**Шаблон:** `section-templates/program-section.tsx`

---

### 5. Блок эксперта (ОБЯЗАТЕЛЬНО)

**Цель:** Доверие через экспертность и личность

**Элементы:**
- [ ] Живое фото (не студийный глянец)
- [ ] Цифры достижений (конкретные)
- [ ] История (2-3 предложения — путь)
- [ ] Регалии (если есть)

```
✅ "Запустил 50+ онлайн-школ с оборотом от 10 млн/мес"
❌ "Эксперт в области маркетинга" (пусто)
```

**Шаблон:** `section-templates/expert-section.tsx`

---

### 6. Социальные доказательства (КРИТИЧНО)

**Видеоотзывы дают +34% конверсии vs текст**

**Требования к отзывам:**
- Видео: до 90 сек, вертикальный формат
- Структура: КТО → БЫЛО → СТАЛО (результат)
- До/После с цифрами

| Сегмент | Количество | Формат |
|---------|------------|--------|
| Средний чек | 6-10 коротких | Карусель/плитка |
| Высокий чек | 3-5 развёрнутых | Полноценные кейсы |

**Шаблон:** `section-templates/testimonials.tsx`

---

### 7. Тарифы и цены (ОБЯЗАТЕЛЬНО)

**Прозрачность = доверие. Рассрочки увеличивают продажи в 3-5 раз.**

| Сегмент | Отображение цены | Рассрочка | Скидки |
|---------|------------------|-----------|--------|
| Средний | Открытая | Да, обязательно | Да |
| Высокий | "От..." или скрыта | По запросу | Нет (эксклюзивность) |

**Структура тарифа:**
- Название (ценностное, не "Базовый")
- Что входит (список с галочками)
- Цена (старая/новая если скидка)
- CTA

**Шаблон:** `section-templates/pricing-section.tsx`

---

### 8. FAQ (ВАЖНО)

**Цель:** Снять страхи и возражения

**Обязательные вопросы:**
1. "Подойдёт ли мне, если я новичок/опытный?"
2. "Сколько времени нужно уделять?"
3. "Что если не получится?"
4. "Как проходит обучение?"
5. "Есть ли рассрочка?"

**Формат:** Accordion (раскрывающиеся ответы)

**Шаблон:** `section-templates/faq-section.tsx`

---

### 9. Футер (ОБЯЗАТЕЛЬНО)

**Юридическая информация:**
- Политика конфиденциальности
- Оферта
- Контакты (email, телефон)
- Реквизиты ИП/ООО

## Средний чек vs Высокий чек

| Параметр | Средний чек (20-50К) | Высокий чек (100К+) |
|----------|---------------------|---------------------|
| **Экраны** | 8-12 | 15-25 |
| **Цель** | Прямая продажа | Запись на диагностику |
| **Отзывы** | Много коротких | 3-5 развёрнутых кейсов |
| **Цена** | Открытая | "От..." или скрыта |
| **Скидки** | Да (таймеры, лимиты) | Нет (эксклюзивность) |
| **Триггеры** | Срочность, ограниченные места | Персональный отбор |
| **CTA** | "Купить", "Записаться" | "Записаться на диагностику" |
| **Дизайн** | Энергичный, динамичный | Quiet Luxury, минимализм |

## Конверсионные элементы

### CTA (Call to Action)

**От первого лица работает лучше:**
```
✅ "Хочу научиться"
✅ "Начать обучение"
✅ "Узнать подробнее"
❌ "Записаться" (безлично)
❌ "Купить" (агрессивно)
```

**Sticky CTA на мобильных** — кнопка всегда видна при скролле.

**Шаблон:** `section-templates/cta-sticky.tsx`

---

### Триггеры срочности

**ВАЖНО: Только реальные!** Фейковые таймеры убивают доверие навсегда.

**Допустимо:**
- Реальная дата старта потока
- Реальное количество мест
- Реальная скидка ранней записи

**Недопустимо:**
- Вечный таймер "осталось 2 часа"
- "Последние 3 места" (когда их 300)

---

### Рассрочки

**Увеличивают продажи в 3-5 раз.** Обязательны для чека от 20К.

**Формат:**
```
✅ "3 000 ₽/мес при рассрочке на 12 месяцев"
✅ "Разбейте на 4 платежа без переплат"
```

---

### Квизы вместо форм

Квизы конвертируют лучше стандартных форм:
- 3-5 вопросов
- Прогресс-бар
- Персонализированный результат

## Дизайн 2026

### Визуальные тренды

| Тренд | Описание | Когда использовать |
|-------|----------|-------------------|
| **Quiet Luxury** | Минимализм, много воздуха | Премиум-сегмент |
| **Кинетическая типографика** | Анимированный текст | Hero-секция |
| **Интерактивный сторителлинг** | Скролл-погружение | Личный бренд |
| **Микроанимации** | Hover-эффекты (+15% кликов) | Везде умеренно |
| **Glassmorphism** | Полупрозрачность + blur | Карточки тарифов |
| **Тёмная тема** | Элегантность | Премиум, tech |

### Типографика

**Комбинация:** Serif (заголовки) + Sans-serif (текст)

```
Заголовки: Didot, Garamond, Playfair Display
Текст: Inter, SF Pro, Montserrat
```

### Цвета

**Правило 2-3 цветов:**
- Нейтральная база (белый/тёмный)
- 1-2 акцентных цвета (signature color бренда)

## Технический чек-лист

**Перед запуском проверь:**

- [ ] **Скорость загрузки < 3 сек** (каждая секунда = -20% конверсии)
- [ ] **Mobile-first** (60%+ трафика с мобильных)
- [ ] **Core Web Vitals в зелёной зоне** (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] **Touch-friendly кнопки** (min 44×44 px)
- [ ] **Sticky CTA на мобильных**
- [ ] **Изображения оптимизированы** (WebP, lazy loading)
- [ ] **Формы работают** (отправка, валидация)
- [ ] **Аналитика подключена** (Яндекс.Метрика + цели)
- [ ] **Pixel Facebook/VK** (если реклама)
- [ ] **SSL сертификат** (https)

## Common Mistakes

| Ошибка | Почему плохо | Как правильно |
|--------|--------------|---------------|
| Нет чёткого УТП в hero | Посетитель уходит за 3 сек | Формула: результат + для кого + за счёт чего |
| Только текстовые отзывы | -34% доверия vs видео | Добавить видеоотзывы |
| Скрытая цена (средний чек) | Раздражает, уходят | Показать цену и рассрочку |
| Фейковые таймеры | Убивает доверие навсегда | Только реальные дедлайны |
| Нет рассрочки | Теряешь 60-80% покупателей | Добавить Тинькофф/Долями |
| Слишком длинный лендинг (средний чек) | Устают скроллить | 8-12 экранов максимум |
| Нет мобильной версии | 60% трафика теряется | Mobile-first подход |
| CTA "Купить" везде | Агрессивно, отталкивает | "Хочу научиться", "Начать" |
| Нет блока эксперта | Нет доверия к продукту | Добавить фото + достижения |
| Размытая ЦА | "Для всех" = ни для кого | 3-4 конкретных портрета |
